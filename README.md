## 简介

蓝湖主题色识别器是一款在蓝湖上显示主题色的Chrome插件

点击图标按钮弹出设置页面，支持自定义色值和名称

<img src="./images/popup.png">

自动识别出主题色色值和名称

<img src="./images/demo.png">
